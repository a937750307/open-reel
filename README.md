# 本地视频编辑器

该项目是基于开源项目[openreel-video](https://github.com/Augani/openreel-video)的汉化处理版本。

是一款功能齐全的基于浏览器的视频编辑器，完全运行在客户端。

采用 React、TypeScript、WebCodecs 和 WebGPU 构建，实现专业级视频剪辑，无需昂贵软件或云处理。

> 开源的 CapCut 替代方案。专业的视频编辑，在你的浏览器中进行。没有上传。没有安装。100%开源。

演示地址：https://vidoeedit.937788.xyz/


## 为什么选择 OpenReel？

- 100%客户端 ——你的视频永远不会离开你的设备。没有上传，没有云端处理，完全隐私。
- 无需安装 ——在 Chrome/Edge 上运行。直接打开开始编辑。
- 专业功能 ——多轨时间线、关键帧动画、调色、音频效果等。
- GPU 加速——WebGPU 和 WebCodec，实现流畅的 4K 编辑和快速导出。
- Free Forever——MIT 授权，无订阅，无水印。

## 特色

### Video Editing
- 多轨时间线 ——无限视频、音频、图像、文本和图形轨道
- 实时预览 ——带 GPU 加速的流畅播放
- 精准编辑 ——帧级精确的擦除、剪切、裁剪、分割、波纹删除
- 过渡 ——交叉淡出、黑白、擦拭、滑音效果
- 视频特效 ——亮度、对比度、饱和度、模糊、锐化、发光、小景画、色度键
- 混合模式 ——乘法、屏幕、叠加、加减等
- 速度控制 - 0.25x 到 4x，保留音高
- 裁剪与变换 ——位置、缩放、旋转，带有 3D 透视

### Graphics & Text
- 专业文本编辑器 ——丰富的样式、阴影、轮廓、渐变
- 20+文本动画 ——打字机、淡入、滑动、弹跳、弹跳、弹性、故障
- 卡拉 OK 风格字幕 ——逐字高亮并同步音频
- 形状工具 ——矩形、圆形、箭头、多边形、带有填充/笔画的星形
- SVG 支持 ——导入带有色彩调和动画的 SVG
- 贴纸与表情符号 ——内置库
- 背景生成器 ——纯色、渐变、网格渐变、图案
- 关键帧动画 ——用 20+缓和曲线为任意属性制作时间动画

### Audio
- 多轨混音 ——无限音轨，实时混音
- 波形可视化 - 视觉音频编辑
- 音频效果—— 均衡器、压缩器、混响、延迟、合唱、镶边器、失真
- 音量与声像 ——每个剪辑的淡入/淡出控制
- 节拍检测 ——自动生成与音乐同步的标记
- 音频低垂 ——对话播放时自动减少音乐
- 降噪—— 三遍降噪（音调、宽带、隆隆）

### Color Grading
- 色轮 ——升光、伽马、增益控制
- HSL 调整 ——色相、饱和度、明度微调
- 曲线编辑器 - RGB 和单个通道曲线
- LUT 支持 ——导入并应用 3D LUT
- 内置预设 ——一键调色

### Export
- MP4（H.264/H.265）——通用兼容性
- WebM（VP8/VP9/AV1）——网页优化格式
- ProRes - 专业中间格式（代理、LT、标准、HQ、4444）
- 高质量预设 - 4K @ 60fps，1080p，720p，480p
- 自定义设置 ——码率、帧率、编解码器选项、色深
- 硬件编码 ——用于快速导出的 WebCodecs
- AI 放大 ——使用 WebGPU 着色器提升分辨率
- 音频导出 - MP3、WAV、AAC、FLAC、OGG
- 图像序列 - JPG、PNG、WebP 帧导出
- 进展跟踪 ——实时进展并支持取消

### Professional Tools
- 无限撤销/重做 ——完整恢复历史
- 自动保存 ——永远不会丢失工作（IndexedDB 存储）
- 键盘快捷键 ——专业工作流程
- 吸附网格 - 磁性对准
- 轨道管理 ——显示/隐藏、锁定/解锁、重新排序
- 字幕支持 ——带可自定义样式的 SRT 导入
- 屏幕录制 ——录制屏幕、摄像头，或两者兼有
- 项目共享 - 导出/导入项目文件

### Performance
- WebGPU 渲染 - GPU 加速合成
- WebCodecs API - 硬件视频解码/编码
- 帧缓存 - LRU 缓存以实现平滑回放
- Web Workers - 后台处理
- 4K 支持 ——编辑并导出 4K 分辨率
